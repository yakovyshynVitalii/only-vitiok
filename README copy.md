# only-vitiok

Автоматизований пайплайн для масового завантаження медіа з AI-підготовкою metadata.

## Що робить

1. Аналізує фото/відео з папки `media`.
2. Генерує `media-config.json` з:
- `hashtags`: до 15 глобальних хештегів.
- `items`: масив медіа (`title`, `description`, `hashtags`, `filePath`).
3. Завантажує медіа на сайт та заповнює поля metadata (якщо поля знайдені у формі).

## Налаштування

1. Створи `.env` на основі `.env.example`.
2. Встанови Ollama.

### Як скачати модель

1. Скачай модель:

```bash
ollama pull llava:7b
```

2. Перевір, що модель доступна локально:

```bash
ollama list
```

3. Перевір `.env`:
- `OLLAMA_URL=<твій_ollama_url>`
- `OLLAMA_MODEL=llava:7b`
- `BASE_URL=<базовий_url_сайту>`
- `CREATE_URL=<url_сторінки_створення>`
4. Зроби авторизацію:

```bash
node login.js
```

## Запуск

Згенерувати конфіг:

```bash
node scripts/generate-config.js
```

Дозволені категорії тегів:
- редагуй файл `tag-categories.txt`
- генератор вибирає теги тільки з цього файлу

Завантажити за конфігом:

```bash
node upload.js
```

Окремий скрипт для додавання тегів (із завершенням роботи та закриттям браузера):

```bash
node add-tags.js
```

## Нотатки

- Для відео аналіз покращується, якщо встановлено `ffmpeg` (скрипт бере кадр з відео).
- Якщо Ollama/модель недоступні, скрипт формує fallback metadata і продовжує роботу.
